# Lora Spa Website

## 🌿 Overview
Lora Spa is a beauty and wellness brand based in Vietnam. This project is a website designed to go beyond simply presenting services—it aims to create a calming and immersive digital experience that reflects the spa’s values of relaxation, professionalism, and holistic care.

The website communicates a sense of tranquility while guiding users through services focused on stress relief, pain reduction, and overall well-being.

---

## ✨ Features
- Clean and calming UI design
- Service and product showcase
- Responsive design (mobile-first approach)
- Smooth and intuitive user experience
- Clear content structure for easy navigation

---

## 🎯 Goal
The goal of this project was to:
- Translate the physical spa experience into a digital environment
- Create a visually soothing interface
- Improve accessibility and usability for potential customers
- Strengthen the brand identity of Lora Spa online

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- GSAP

---

## 📱 Responsive Design
This website is designed with a mobile-first approach and adapts seamlessly across:
- Mobile devices
- Tablets
- Desktop screens

---

## 🎨 Design Approach
The design focuses on:
- Soft color palettes
- Minimal and elegant layouts
- Clear typography
- A balance between aesthetics and usability
